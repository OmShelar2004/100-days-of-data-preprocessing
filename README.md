# 100 Days of Data Preprocessing

This repository documents my journey of learning Data Preprocessing for Data Science and Machine Learning using Python and Pandas.

## Topics Covered

- Data Inspection
- Missing Value Handling
- Duplicate Removal
- Outlier Detection
- Data Cleaning
- Category Standardization
- Encoding
- Date Processing
- Feature Engineering
- Preparing Data for Machine Learning

## Goal

Build strong data handling skills before moving into Machine Learning, Deep Learning, and RAG systems.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
